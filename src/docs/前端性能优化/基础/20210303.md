# 1. 从输入url到页面展现发生了什么
+ DNS解析： 将域名解析成IP地址
+ TCP连接： TCP三次握手
+ 发送Http请求
+ 服务器处理请求并返回http报文
+ 浏览器解析渲染页面
+ 断开连接： TCP四次挥手
